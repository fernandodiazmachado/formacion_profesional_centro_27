Ejercicio02.- (Aplicar trabajo modular utilizando funciones). En C, JAVA, Javascript o Python Hacer una calculadora, cuando arranque el programa tiene que aparecer el siguiente menú:
a-Sumar
b-Restar
c-Multiplicar
d-Dividir
e-Salir

Hacer una función para cada operación, que lea los números ingresados y devuelva el resultado correspondiente. El menú tiene que aparecer después de cada operación hasta que el usuario elija la opción e.
